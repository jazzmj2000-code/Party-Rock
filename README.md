# Party-Rock
Daily Task Scheduler
Overview of Project 
In this project I'm using PartyRock to develop a 'Daily Task Scheduler Application' aimed at enhancing our productivity and organization. For those not familiar with PartyRock, it's a new platform offered by AWS designed to simplify app development through the use of generative AI. I've had a lot of fun playing around with PartyRock recently, so I made this.

Steps to be performed:

Generating the daily task scheduler application
Changes in existing widgets
Adding new widgets
Publishing your app
